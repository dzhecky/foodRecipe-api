# Food Recipe API

API for food recipe app build with Express JS and using PostgreSQL for handle database.

## Instalation

- git clone https://github.com/dzhecky/foodRecipe-api.git
- cd foodRecipe-api / (open on your code editor)
- npm install
- nodemon start
